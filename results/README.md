# CBB Results

## Six-Run Experiment

CBB was run six times across two prompt conditions:

**Condition A - Baseline (Runs 1-3)**
Standard prompt asking the model to answer and state confidence.

**Condition B - Uncertainty-Pressured (Runs 4-6)**
Explicit instruction to consider whether the model might be 
confused, have conflicting information, or be at the limits 
of its knowledge — before predicting and answering.

## Key Finding

Error Recall = 0.000 in all six runs across both conditions.
Overconfidence is prompt-resistant, it cannot be fixed by 
telling the model to doubt itself.
